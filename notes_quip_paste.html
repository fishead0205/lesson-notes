Deep Dive into LLMs

How pretraining data is prepared

  • Start with large web datasets like Common Crawl.
  • Filtering steps: blocklist bad sites, keep high-quality text, remove personal data, normalize text.

Tokenization / Symbol Conversion

  • Byte-level representation: encode text as bytes (0-255).
  • Byte Pair Encoding (BPE): merge frequent symbol pairs to form useful tokens.
  • Token: a chunk of text the model reads as one unit.
  • LLM: Large Language Model trained on huge text corpora.

Flow (same meaning)

Raw Web Pages -> Filtering -> Clean Text -> Bits (0/1) -> Bytes (0-255) -> BPE Tokens -> Train LLM
